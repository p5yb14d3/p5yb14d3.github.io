DOWNLOAD FOR WINDOWS

Prep your folders

Organize or categorize your videos into the folders you like. For folders to show as an image, place an image inside the folder and name the image the same as the folder containing it.

Videodrome supports multiple levels of sub-directories.

Install & generate

Download the videodrome.exe file from above, install it onto your PC. Once installed, open the windows command prompt, and key in videodrome generate <path_to_your_videos> e.g. videodrome generate ~\videos.

If you've Python installed, you can install Videodrome using pip.
$ pip install videodrome

Ready to rock!

Videodrome will launch the media center in your web browser immediately after generating. You can bookmark it or set it as your homepage on your browser for easy access.

Subsequent access to the page requires no regenerating (unless you wish to add new videos to your folders and would like them to show up on your media center).

Fully Responsive

This theme will look great on any device, no matter the size!

Run anywhere

Once you have generated your media center, you can port your entire video folder to any platform that supports a browser with access to your video folders.

Fully Customizable

With a little knowledge of CSS and HTML, you can customize the look and feel of your media center by just changing the files located under _data/themes

Navigate with these input devices with the built-in Keynav.js

Keyboard

Use the arrow keys to navigate. Enter to select. Esc to go back.

Mouse

Or you can choose to use the good old mouse. Simply point and click!

TV Remote Control

You can also navigate by plugging in this USB-enabled remote control. Get one from Amazon.
